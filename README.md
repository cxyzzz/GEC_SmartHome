# GEC_smart_home

GEC智能家居生产实习。使用的设备为 GEC6818 arm 开发板，开发环境为 Ubuntu 14.04

## 0x01 复习 C 语言基础

1. 求两个数的最大值
2. 判断奇偶
3. 判断素数
4. 使用函数交换两个数的值
    - 指针的使用

## 0x02 I/O 练习

1. 判断是否为闰年
2. 判断是否为水仙花数
3. 打开一个文件。输出其所有内容
4. 交换两个文件的内容
5. 将屏幕显示为某一颜色
    - 将文件映射到内存，加快读写速度。

## 0x03 在屏幕绘制指定图案

1. 画一个矩形
2. 画一个圆
3. 在任意位置画一个任意长度的矩形

## 0x04 在屏幕上显示字符

1. 在屏幕上显示自己的名字和学号
2. 任意输入一个数字，在屏幕上显示

## 0x05 在屏幕上显示位图
